MAD9022-W21

Cross Platform Application Development II

Get Started →

JavaScript based Web Apps

Learn how to use JavaScript, CSS3 and HTML5 to create both Single Page Applications and Progressive Web Apps.

JavaScript in Production

Apply your JavaScript knowledge to work with many HTML5 APIs and a variety of production tools like Webpack, GitHub, Git, Babel, and others.

Hybrid Apps

Leverage all your HTML, CSS, and JavaScript skills to create both interactive hybrid mobile apps. Use AJAX technology to dynamically update the data in your app. Use Cordova to package your apps to run on Android or iOS devices.
